# SENPAI - AI Assistant

シンプルで使いやすいAIアシスタントアプリケーション。画面のスクリーンショットを撮影し、AIが画面を見て質問に答えます。

## 🎯 特徴

- **📸 スクリーンショット分析** - 画面をキャプチャしてAIが内容を理解
- **💬 自然な会話** - テキストまたは音声で質問
- **🔊 音声回答** - AIの回答を音声で読み上げ（ON/OFF可能）
- **🎤 音声入力** - マイクボタンで音声入力
- **⌨️ キーボード操作** - Returnキーで質問送信
- **📝 履歴表示** - 過去の質問と回答を表示

## 🚀 使い方

### 1. 起動

```bash
python run_assistant.py
```

### 2. 基本操作

1. **スクリーンショットを撮る**
   - 「📸 スクリーンショット」ボタンをクリック
   - 画面全体がキャプチャされます

2. **質問する**
   - テキスト入力欄に質問を入力
   - Returnキーを押して送信
   - または🎤マイクボタンで音声入力

3. **回答を聞く**
   - AIが画面を見て回答
   - 「🔊 音声回答」がONの場合、音声で読み上げ

## ⚙️ 設定

### 音声回答のON/OFF
- 「🔊 音声回答」チェックボックスで切り替え
- OFFにすると、テキストのみで回答

### AIモデルの変更
`ai_module_improved.py`の223行目付近で変更可能：

```python
# デフォルト: gpt-4o-mini（高速・低コスト）
model="gpt-4o-mini"

# より高精度な回答が必要な場合:
# model="gpt-4o"
# model="gpt-4.1-mini"
# model="gpt-4.1"
```

### TTS音声の変更
`tts_module.py`で音声タイプを変更可能：

```python
# デフォルト: nova（女性的な声）
voice="nova"

# その他の選択肢:
# voice="alloy"   # 中性的
# voice="echo"    # 男性的
# voice="fable"   # 英国風
# voice="onyx"    # 深い男性的
# voice="shimmer" # 明るい女性的
```

## 🔧 トラブルシューティング

### 音声入力が動作しない
- マイクの接続を確認
- マイクの権限を許可
- PyAudioが正しくインストールされているか確認

### 音声出力が動作しない
- スピーカー/ヘッドフォンの接続を確認
- 音量設定を確認
- pygameが正しくインストールされているか確認

### API エラー
- OPENAI_API_KEY環境変数が設定されているか確認
- APIキーが有効か確認
- インターネット接続を確認
